# Rust Firefly III

Rust OpenAPI client for [Firefly III](https://firefly-iii.org/).

## Installation

```shell
cargo add firefly-iii
```

## Documentation

Documentation for is available on [Docs.rs](https://docs.rs/firefly-iii/latest/firefly_iii/).
